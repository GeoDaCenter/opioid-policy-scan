1.The table key is GEOID and COUNTYFP, which is combined by STATEFP (2 digits) + COUNTYFP (3 digits) 

2."GetVeteran_CountyLevel" is a R script to read local veteran file, calculating correct GEOID(table key) and do related calculation

3."Merged_C&T_Code.py" is a Python script to make county level data more comprehensive by adding variables which tract level data has while county level one does not have

4."original_county_level_data" and "Original Data Dictionary for County Level Data" is merged table only including county level data from OEPS without data from tract level data

5."T&C_Merged_Table" is the result of "Merged_C&T_Code.py"
